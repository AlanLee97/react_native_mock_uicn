# 输入框组件



## 参数

| 参数           | 类型   | 说明       | 可选值   | 默认值  | 是否必传 |
| -------------- | ------ | ---------- | -------- | ------- | -------- |
| type           | String | 输入框类型 | password | 无      | 否       |
| primaryColor   | String | 主题颜色   |          | #409EFF | 否       |
| selectionColor | String | 光标颜色   |          | #409EFF | 否       |
